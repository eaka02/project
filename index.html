Knife Guide Arthritis Aid

Cost Estimate $0~10

Description

The Knife Guide Arthritis Aid is a handle that attaches to the back of a knife that allows a user to use both hands and requires less grip strength for the same amount of control.

Disability Type

This device is intended to support people with arthritis when cutting with a larger kitchen knife.

purpose

The Knife Guide Arthritis Aid is a handle that attaches to the back of a knife that allows a user to use both hands while cutting and requires less grip strength for the same amount of control.

How To Use

To use, place the device over the back of the knife.

The new handle can now be gripped at a more natural angle than the original handle, and the knife used as normal.

However, at least one hand must remain on the original handle during use. Attempting to hold the knife with just the new handle could cause the knife to fall out of the handle.

Infromation for makers

Skills required:

Basic 3D printing skills, basic assembly skills

3 hours

tools needed:

Build instructions:

This device consists of a single 3D-printed part with no assembly required
